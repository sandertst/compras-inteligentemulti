# Compras Inteligente - Sistema para vender

Versão multiusuário do app, pronta para GitHub Pages.

## O que foi adicionado
- tela inicial com código de acesso
- cada cliente entra com o próprio código
- o mesmo sistema atende vários clientes
- 73 produtos da planilha
- edição completa dos itens
- filtro por categoria
- modo supermercado rápido
- exportação da lista
- PWA instalável
- sincronização opcional via Firebase
